Detection

Bubble detection threshold
Minimum confidence for the detector to accept a bubble. Lower → more bubbles detected (including noise). Higher → fewer, more reliable. Default: 0.50.
SFX detection threshold
Second-pass threshold for free-floating text (SFX / sound effects). Should be lower than the main threshold to catch small SFX not found in the first pass. Default: 0.30.
Min bubble area (px²)
Bubbles smaller than this area (in pixels²) are discarded after overlap clipping. Increase to ignore tiny false-positive detections. Default: 400.

Rendering

Max font size (px)
Upper limit for automatic font size selection. The auto-fit algorithm finds the largest font that fits the bubble; raising this cap lets text grow larger in big bubbles. Default: 90.
Inpaint mask shrink (px)
Number of pixels to pull the inpainting mask inward from the bubble border. Prevents LaMa from accidentally inpainting outside the bubble edge. Default: 1.

Translation

Bubbles per LLM request
How many speech bubbles are sent to the LLM in one batch. Larger batches provide more context (good for gender/tone consistency within a page) but make JSON parsing less reliable. Default: 5.
LLM retries on failure
Number of times to retry a failed LLM call (bad JSON, empty response, timeout). Higher values make translation more robust on unstable models at the cost of extra time. Default: 3.

Connection

Ollama API URL
Full URL to the Ollama generate endpoint. Change this if Ollama is running on a different host or port. Default: http://localhost:11434/api/generate.

Diagnostics

Debug boxes on result
Draw colored bounding boxes on translated pages to diagnose detection and OCR results.
Green — translated successfully
Blue — SFX translated
Orange — OCR found text, translation failed
Red — no OCR text
Verbose LLM logging (server console)
When enabled, every prompt sent to the LLM and every response from it is printed to the server console. Useful for diagnosing translation issues. Slows things down and produces a lot of output.
Mask debug
Save text segmentation mask for each bubble as mask_debug_N.png in the results folder. Green overlay = detected text pixels.
